# Web Security Baseline Scanner

Automated Web Security Baseline Scanner for detecting common security misconfigurations and generating professional security reports.

## Overview

The Web Security Baseline Scanner is a lightweight security assessment tool designed to quickly identify common security issues in web applications. It performs automated checks and generates a professional PDF report for easy review.

This tool is useful for:
- Security engineers
- Penetration testers
- Developers
- Security auditors

---

## Features

- Automated web security checks
- Security header analysis
- Basic vulnerability detection
- Clean and simple UI
- PDF security report generation
- Scan history dashboard

---

## Technologies Used

- Node.js
- Express.js
- HTML
- CSS
- JavaScript
- PDFKit

## Installation

Clone the repository:

```
git clone https://github.com/AkshayUSalunke/web-security-baseline-scanner.git
```

Go into the project directory:

```
cd web-security-baseline-scanner
```

Install dependencies:

```
npm install
```

Start the server:

```
node server.js
```

Open the browser and go to:

```
http://localhost:3000
```

---

## How It Works

1. Enter the target website URL
2. The scanner performs security baseline checks
3. Detected issues are analyzed
4. A professional PDF report is generated

---

## Example Checks

The scanner currently performs checks for:

- Missing Security Headers
- Basic configuration weaknesses
- Web security baseline misconfigurations
